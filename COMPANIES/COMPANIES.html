Disclaimer

Process and references

  • The data was downloaded from the web page of Superintendencia Financiera de Colombia.
  • Each company publishes its financial results every quarter, and the information presented here is extracted directly from those official reports.
  • The data processing was performed in Excel, the web page was built using HTML and CSS, and the interactive application was developed in Python with Dash and deployed on Render.

Duration
